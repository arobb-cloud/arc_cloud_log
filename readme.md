
		# Cloud Log Monitoring Tool

		Python-based operational monitoring project designed to simulate real-world cloud and enterprise production support workflows.

		## Features

		- Parses application log files
		- Detects ERROR and WARNING events
		- Generates operational metrics
		- Simulates incident monitoring workflows

		## Technologies

		- Python 3
		- Git/GitHub
		- VS Code

		## Future Enhancements

		- CloudWatch integration
		- Datadog webhook alerts
		- Docker containerization
		- Terraform deployment
		- Jenkins CI/CD pipeline
		- Kubernetes deployment
